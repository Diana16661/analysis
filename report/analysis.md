У цьому звіті виконано порівняльний аналіз трьох платформ:
- Android
- iOS
- Web (браузер)
Метою є оцінка архітектури, безпеки, доступу до ресурсів та придатності для розробки.
📱 2. Загальна характеристика платформ
Android
Тип: мобільна платформа
Призначення: споживчі пристрої
Пристрої: смартфони, планшети, TV
iOS
Тип: мобільна платформа
Призначення: premium споживчі пристрої
Пристрої: iPhone, iPad
Web
Тип: web-платформа
Призначення: універсальні додатки
Пристрої: ПК, смартфони, планшети
⚙️ 3. Архітектура та середовище виконання
Android
ОС: Android (на базі Linux)
Ядро: Linux kernel
Архітектура: ARM, x86
Runtime: ART (Android Runtime)
iOS
ОС: iOS
Ядро: Darwin (XNU)
Архітектура: ARM
Runtime: Native (Swift/Objective-C)
Web
ОС: не залежить
Ядро: браузер
Архітектура: будь-яка
Runtime: JavaScript engine (V8, SpiderMonkey)
